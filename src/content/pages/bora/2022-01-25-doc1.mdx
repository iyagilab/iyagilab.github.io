### 💜 BORA 💜 서비스 링크 바로가기 넣기

개인이 수집한 웹 링크데이터에 대한 키워드와 요약 정보를 제공함으로써 일정 시간이 흐른 뒤 자료를 다시 확인하고 싶거나 많은 양의 데이터에서 원하는 정보의 빠른 확인이 필요할 때 수월하게 데이터를 확인하고 활용할 수 있는 서비스이다.

### ✨ 주요 기능 ✨ (나중에 사진 첨부 예정)

**🔑 데이터 요약 및 다중 문서정리**

- 사용자가 수집한 데이터(웹 링크, PDF 문서 등)를 기반으로 요약 데이터를 제공하여 사용자가 데이터의 원문을 확인하기 이전에 미리 그 데이터의 정보를 쉽게 파악할 수 있다.
- 다중 문서 정리 기능으로 수집이 끝난 비슷한 카테고리의 여러 데이터에 대해 관계 분석을 수행하여 중복되는 내용을 제거하고, 합쳐진 데이터를 기준으로 요약정리 내용을 제공함으로써 많은 양의 데이터를 효과적으로 확인할 수 있도록 한다.

**🔑 데이터 미리보기 (이미지로 어떤 글인지 떠올릴수 있다)**

- 스크롤 다운과 같은 효과를 적용한 데이터 미리보기를 제공하여 원본을 확인하지 않고도 대략적인 이미지로 데이터를 확인할 수 있는 방법을 제공한다.
- 데이터 수집 및 정리 이후 요약정보나 미리보기를 확인하는 과정에서 세부 내용의 확인이 필요한 경우는 별도의 다운로드 없이 본문 내용을 확인할 수 있는 전용 뷰어를 제공하거나 원본 페이지로 연결해준다.

**🔑 키워드 추출 및 사용자 히스토리 분석**

- 키워드를 통한 카테고리 추출을 통해 사용자가 보유하고 있는 데이터의 종류와 데이터에 많이 사용되고 있는 키워드들이 어떤 것들이 있는지 미리 제공하여 자신의 키워드를 평소에 인지할 수 있도록 한다.

### 🧑🏻‍💻 사용자 UI 기능 🧑🏻‍💻

- 컨텐츠 / pdf / 파일 등록
- 타일 / 서머리 뷰 제공
- 상세 페이지에서 코멘트 등록으로 추가 메모 또는 다른 사용자 간의 커뮤니 케이션 제공
- 사용자 분석 및 기타 기능 제공
- 게시글 공개(로그인안해도보임) & 비공개(나만볼수있음) 기능 제공
- 무한스크롤 & 더보기 기능 제공
- 좋아요 기능 제공
- 컨텐츠 검색 제공 ( searchIndex 사용)
